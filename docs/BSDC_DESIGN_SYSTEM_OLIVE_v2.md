# BSDC Design System — Olive Edition v2

## 1. Overview

This document defines the single visual system for BSDC frontend.

It replaces all previous industrial / HUD / cinematic variants with a unified editorial design language.

---

## 2. Color System

### Base
- #0F1411 → Primary background

### Primary Accent (Olive System)
- #8A9A86 → Primary interactive accent

### Light Surface
- #F5F3EF → Text, highlights, contrast surfaces

### Dark Accent
- #4A5343 → Secondary depth tone

---

## 3. Typography

### Headings
- Font: Space Grotesk
- Weight: 300 ONLY
- No exceptions

### Body / UI
- Font: Geist Mono
- Weight: 300–400 max

### Rules
- No bold hierarchy
- No semibold usage
- No black weights
- Hierarchy achieved via:
  - size
  - spacing
  - opacity
  - tracking

---

## 4. Design Principles

### Forbidden styles
- Industrial UI
- HUD overlays
- Sci-fi theming
- Neon accents
- Multi-color systems

### Allowed style direction
- Editorial architecture
- Minimal contrast systems
- Calm cinematic spacing
- Material-like depth layering

---

## 5. Interaction Language

- GSAP only (if motion used)
- No Framer Motion
- No physics randomness
- Smooth, slow transitions

---

## 6. Visual Identity

The system should feel like:

"architectural field documentation presented as a luxury editorial interface"

---

## 7. Migration Rule

All existing components must be migrated to this system without introducing new colors or new typography rules.
