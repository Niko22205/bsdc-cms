# BSDC CMS Project

## Goal
Rebuild company website bsdc.bg as a custom CMS without WordPress.

## Stack
- Next.js
- TypeScript
- Tailwind CSS
- PostgreSQL
- Prisma

## Features
- Pages
- Services
- Blog
- Media
- Menu
- SEO
- Admin dashboard

## Rules
- Clean architecture
- No unnecessary dependencies
- No plugin-based CMS
- Keep code modular
