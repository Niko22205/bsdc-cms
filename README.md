# BSDC Project

Custom CMS and company website for bsdc.bg.

## Goal

Replace the existing broken WordPress website with a lightweight custom CMS.

## Current status

Initial project structure prepared.

## Planned stack

- Next.js
- TypeScript
- Tailwind CSS
- PostgreSQL
- Prisma
- Custom admin dashboard
- Public company website

## Project rules

- No WordPress
- No plugin-based CMS
- Keep dependencies minimal
- Keep code clean and modular
- Prioritize speed, SEO, and maintainability
